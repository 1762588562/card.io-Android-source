card.io for Android example
===========================

This sample shows how to integrate card.io in Android.

To give it a try:

## With Gradle (modern)

1. Clone this repo.
2. `$ cd SampleApp`
3. `$ android update project -p .`
4. Plug in your device
5. `$ ./gradlew installDebug`
6. Run the demo app!

## With Ant (historic)

1. Clone this repo.
2. `$ cd SampleApp`
3. `$ android update project -p .`
4. Plug in your device
5. `$ ant debug install`
6. Run the demo app!

That's it!!
